Este é um projeto de modernização do site da **London Seguradora**. O objetivo foi transformar a presença digital tradicional em uma experiência web de alta performance, com design "Premium Dark Mode" e interatividade fluida.

## ✨ Características

- 🌑 **Premium Dark Mode**: Estética sofisticada inspirada em Raycast e Framer.
- 🍱 **Bento-box Layout**: Organização modular e moderna de serviços e informações.
- 🧊 **Glassmorphism**: Efeitos de transparência e desfoque de fundo para profundidade visual.
- ⚡ **Alta Performance**: Desenvolvido com Vite para carregamento instantâneo.
- 📱 **Totalmente Responsivo**: Experiência otimizada para Desktop, Tablets e Mobile.
- 🎭 **Animações Fluídas**: Micro-interações alimentadas por Framer Motion.

## 🛠️ Tecnologias Utilizadas

- **React 19**: Biblioteca para construção da interface.
- **TypeScript**: Tipagem estática para robustez e produtividade.
- **Tailwind CSS v4**: Otimização de estilos e design system moderno.
- **Framer Motion**: Motor de animações declarativas.
- **Lucide React**: Biblioteca de ícones minimalistas e consistentes.

## 🚀 Como Executar

### Pré-requisitos
- [Node.js](https://nodejs.org/) (versão 18 ou superior)
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)

### Instalação
1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/london.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
